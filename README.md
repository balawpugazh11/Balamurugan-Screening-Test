# Tandemloop Screening Test – Coding Solutions by Balamurugan

## Project Description
This repository contains my solutions for the Full Stack Developer screening test sent by Tandemloop Technologies.  
There are four coding problems, each solved using simple and clear logic in JavaScript.

## Problems Covered
1. **Calculator**  
   Performs addition, subtraction, multiplication, and division using a class.

2. **Odd Numbers Series**  
   Prints the first ‘a’ odd numbers based on user input.

3. **Custom Series**  
   Generates a series as described in the test instructions.

4. **Multiples Counter**  
   Counts how many list elements are multiples of numbers 1 to 9.

## How to Run
- Clone or download this repository.
- Open each `.js` file (problem1.js, problem2.js, etc.) in your local editor.
- Run using Node.js or any online JavaScript compiler.
- Test the functions with sample inputs to see the output.

## Technologies Used
- JavaScript (ES6)
- Node.js (for running scripts, optional)
